## 🔄 CRUD OPERATIONS FLOW

```
CREATE:
User → DataForm → createData() → addDoc() → Firestore → Success Message

READ:
User → DataList → getAllData() → getDocs() → Firestore → Display Data

UPDATE:
User → DataList (edit) → updateData() → updateDoc() → Firestore → Refresh List

DELETE:
User → DataList (delete) → deleteData() → deleteDoc() → Firestore → Refresh List
```
